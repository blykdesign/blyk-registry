# Badge

Displays a badge or a component that looks like a badge.

## Installation

```bash
npx shadcn@latest add "https://registry.blyk.design/r/base/badge.json"
```

## Usage

```tsx
import { Badge } from "@/components/ui/badge"
```

```tsx
<Badge variant="outline">Badge</Badge>
```
