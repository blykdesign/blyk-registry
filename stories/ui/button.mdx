# Button

Displays a button or a component that looks like a button.

## Installation

```bash
npx shadcn@latest add "https://registry.blyk.design/r/base/button.json"
```

## Usage

```tsx
import { Button } from "@/components/ui/button"
```

```tsx
<Button variant="outline">Button</Button>
```
