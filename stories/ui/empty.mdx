# Empty

A placeholder component for empty states.

## Installation

```bash
npx shadcn@latest add "https://registry.blyk.design/r/base/empty.json"
```

## Usage

```tsx
import { Empty } from "@/components/ui/empty"
```

```tsx
<Empty>No results found.</Empty>
```
