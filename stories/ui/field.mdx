# Field

Combine labels, controls, and help text to compose accessible form fields.

## Installation

```bash
npx shadcn@latest add "https://registry.blyk.design/r/base/field.json"
```

## Usage

```tsx
import { Field } from "@/components/ui/field"
```

```tsx
<Field>
  <Label>Email</Label>
  <Input type="email" />
</Field>
```
