# IndicatorCard

<ThemeBadge
  theme="DF Imóveis"
  description="This component belongs to the DF Imóveis theme."
/>

A dashboard indicator card designed to display big numbers and key metrics with an optional trend badge.

## Installation

```bash
npx shadcn@latest add "https://registry.blyk.design/r/df-imoveis/indicator-card.json"
```

## Usage

```tsx
import { IndicatorCard } from "@/components/ui/indicator-card"
import { UserPlus, TrendingUp } from "lucide-react"

<IndicatorCard
  icon={<UserPlus />}
  label="Novos Leads"
  value="12"
  badge={<><TrendingUp /> +3 hoje</>}
/>
```

<Tip>
  Switch to the <strong>DF Imóveis</strong> theme using the toolbar above to see this component with the correct brand colors applied.
</Tip>
