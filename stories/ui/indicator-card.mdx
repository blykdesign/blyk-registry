# IndicatorCard

A dashboard indicator card designed to display big numbers and key metrics with an optional trend badge.

Built as a compound component following the same pattern as `Card`, giving you full control over the layout and composition.

## Installation

```bash
npx shadcn@latest add "https://registry.blyk.design/r/base/indicator-card.json"
```

## Usage

```tsx
import {
  IndicatorCard,
  IndicatorCardIcon,
  IndicatorCardLabel,
  IndicatorCardValue,
  IndicatorCardNumber,
  IndicatorCardBadge,
} from "@/components/ui/indicator-card"
```

```tsx
<IndicatorCard>
  <IndicatorCardIcon>
    <UserPlus />
  </IndicatorCardIcon>
  <IndicatorCardLabel>Novos Leads</IndicatorCardLabel>
  <IndicatorCardValue>
    <IndicatorCardNumber>12</IndicatorCardNumber>
    <IndicatorCardBadge>
      <TrendingUp /> +3 hoje
    </IndicatorCardBadge>
  </IndicatorCardValue>
</IndicatorCard>
```

## Sub-components

| Component | Description |
|---|---|
| `IndicatorCard` | Root container with border, shadow, and hover/active states |
| `IndicatorCardIcon` | 32x32 icon area in muted foreground color |
| `IndicatorCardLabel` | Descriptive label (text-sm, muted) |
| `IndicatorCardValue` | Flex row that holds the number and optional badge |
| `IndicatorCardNumber` | Big number display (text-3xl, semibold) |
| `IndicatorCardBadge` | Optional outline badge with icon and text |
