# Toast

A succinct message that is displayed temporarily.

## Installation

```bash
npx shadcn@latest add "https://registry.blyk.design/r/base/toast.json"
```

## Usage

```tsx
import { useToast } from "@/components/ui/toast"
```

```tsx
const { toast } = useToast()
toast({ title: "Success" })
```
