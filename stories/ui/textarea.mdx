# Textarea

Multi-line text input. Use with Form and Label for accessible forms.

## Installation

```bash
npx shadcn@latest add "https://registry.blyk.design/r/base/textarea.json"
```

## Usage

```tsx
import { Textarea } from "@/components/ui/textarea"
```

```tsx
<Textarea placeholder="Type your message here." />
```
