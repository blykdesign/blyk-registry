# ImovelCard

<ThemeBadge
  theme="DF Imóveis"
  description="This component belongs to the DF Imóveis theme."
/>

A real estate property card designed to display a property listing with image, highlight tags, price, address, and feature badges.

## Installation

```bash
npx shadcn@latest add "https://registry.blyk.design/r/df-imoveis/imovel-card.json"
```

## Usage

```tsx
import { ImovelCard } from "@/components/ui/imovel-card"
import { Star, Bed, Bath, Car, Maximize2 } from "lucide-react"

<ImovelCard
  image={{ src: "/photo.jpg", alt: "Foto do imóvel" }}
  tags={[{ icon: <Star />, label: "Destaque" }]}
  price="R$ 500.000"
  address="SQS 308 Bloco A, Apartamento 302 - Asa Sul"
  features={[
    { icon: <Bed />, label: "3" },
    { icon: <Bath />, label: "2" },
    { icon: <Car />, label: "1" },
    { icon: <Maximize2 />, label: "200m²" },
  ]}
/>
```

<Tip>
  Switch to the <strong>DF Imóveis</strong> theme using the toolbar above to see this component with the correct brand colors applied.
</Tip>
