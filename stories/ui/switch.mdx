# Switch

A control that allows the user to toggle between a checked and not checked state.

## Installation

```bash
npx shadcn@latest add "https://registry.blyk.design/r/base/switch.json"
```

## Usage

```tsx
import { Switch } from "@/components/ui/switch"
```

```tsx
<Switch />
```
