# Native Select

Displays a native select element.

## Installation

```bash
npx shadcn@latest add "https://registry.blyk.design/r/base/native-select.json"
```

## Usage

```tsx
import { NativeSelect } from "@/components/ui/native-select"
```

```tsx
<NativeSelect>
  <option value="apple">Apple</option>
  <option value="banana">Banana</option>
</NativeSelect>
```
