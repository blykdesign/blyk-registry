# Sidebar

Composable, themeable sidebar with collapsible modes (offcanvas, icon). Use `SidebarProvider` at the app root.

## Installation

```bash
npx shadcn@latest add "https://registry.blyk.design/r/base/sidebar.json"
```

## Usage

```tsx
import { SidebarProvider, Sidebar, SidebarContent, SidebarGroup, SidebarGroupLabel, SidebarGroupContent, SidebarMenu, SidebarMenuItem, SidebarMenuButton, SidebarTrigger, SidebarInset } from "@/components/ui/sidebar"
```

```tsx
<SidebarProvider>
  <Sidebar>
    <SidebarContent>...</SidebarContent>
  </Sidebar>
  <SidebarInset>
    <main>Content</main>
  </SidebarInset>
</SidebarProvider>
```
