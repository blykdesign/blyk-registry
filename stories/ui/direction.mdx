# Direction

Provides a direction context for right-to-left (RTL) support.

## Installation

```bash
npx shadcn@latest add "https://registry.blyk.design/r/base/direction.json"
```

## Usage

```tsx
import { DirectionProvider } from "@/components/ui/direction"
```

```tsx
<DirectionProvider dir="rtl">
  {children}
</DirectionProvider>
```
