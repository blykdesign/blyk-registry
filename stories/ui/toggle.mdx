# Toggle

A two-state button that can be either on or off.

## Installation

```bash
npx shadcn@latest add "https://registry.blyk.design/r/base/toggle.json"
```

## Usage

```tsx
import { Toggle } from "@/components/ui/toggle"
```

```tsx
<Toggle>Toggle</Toggle>
```
