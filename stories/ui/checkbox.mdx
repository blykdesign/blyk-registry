# Checkbox

A control that allows the user to toggle between checked and not checked.

## Installation

```bash
npx shadcn@latest add "https://registry.blyk.design/r/base/checkbox.json"
```

## Usage

```tsx
import { Checkbox } from "@/components/ui/checkbox"
```

```tsx
<Checkbox />
```
