# Item

A flexible list item component for displaying structured content.

## Installation

```bash
npx shadcn@latest add "https://registry.blyk.design/r/base/item.json"
```

## Usage

```tsx
import { Item } from "@/components/ui/item"
```

```tsx
<Item>Item content</Item>
```
