# Spinner

Displays a loading spinner animation.

## Installation

```bash
npx shadcn@latest add "https://registry.blyk.design/r/base/spinner.json"
```

## Usage

```tsx
import { Spinner } from "@/components/ui/spinner"
```

```tsx
<Spinner />
```
