# SuperButton

A larger, bolder button designed for high-impact CTAs, hero sections, and primary actions that need extra visual prominence.

Compared to the standard `Button`, the `SuperButton` features a taller height (56px), larger typography (16px bold), and a more generous border radius.

## Installation

```bash
npx shadcn@latest add "https://registry.blyk.design/r/base/super-button.json"
```

## Usage

```tsx
import { SuperButton } from "@/components/ui/super-button"
```

```tsx
<SuperButton variant="outline">Get Started</SuperButton>
```
