# Input Group

Groups an input with addons like prefixes, suffixes, or buttons.

## Installation

```bash
npx shadcn@latest add "https://registry.blyk.design/r/base/input-group.json"
```

## Usage

```tsx
import { InputGroup } from "@/components/ui/input-group"
```

```tsx
<InputGroup>
  <Input placeholder="Search..." />
</InputGroup>
```
