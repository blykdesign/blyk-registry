# Combobox

Autocomplete input with searchable options. Combines Command and Popover for type-ahead selection.

## Installation

```bash
npx shadcn@latest add "https://registry.blyk.design/r/base/combobox.json"
```

## Usage

```tsx
import { Combobox } from "@/components/ui/combobox"
```

```tsx
<Combobox />
```
