# Button Group

Groups multiple buttons together with shared borders.

## Installation

```bash
npx shadcn@latest add "https://registry.blyk.design/r/base/button-group.json"
```

## Usage

```tsx
import { ButtonGroup } from "@/components/ui/button-group"
```

```tsx
<ButtonGroup>
  <Button>Left</Button>
  <Button>Center</Button>
  <Button>Right</Button>
</ButtonGroup>
```
