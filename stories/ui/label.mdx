# Label

Renders an accessible label associated with controls.

## Installation

```bash
npx shadcn@latest add "https://registry.blyk.design/r/base/label.json"
```

## Usage

```tsx
import { Label } from "@/components/ui/label"
```

```tsx
<Label htmlFor="email">Email</Label>
```
