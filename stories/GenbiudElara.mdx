## Installation

<Step number={1} title="Install the theme components">
  Use the shadcn CLI to install components from the Genbiud Elara registry path.
</Step>

```bash
npx shadcn@latest add button --registry https://registry.blyk.design/r/genbiud-elara
npx shadcn@latest add --all --registry https://registry.blyk.design/r/genbiud-elara
```

<Step number={2} title="Add the theme CSS">
  Copy the Genbiud Elara token overrides into your project. These override the base blyk tokens with the brand palette.
</Step>

```css
[data-theme="genbiud-elara"] {
  --primary: 4 66% 41%;
  --primary-foreground: 6 100% 98%;
  --font-sans: "Barlow", ui-sans-serif, system-ui, sans-serif;
  --font-mono: "Geist Mono", ui-monospace, monospace;
  /* ... see registry/genbiud-elara/tokens/ for the full list */
}
```

<Step number={3} title="Activate the theme">
  Apply the theme using a data attribute on the root element.
</Step>

```html
<html data-theme="genbiud-elara">
```
