## Installation

<Step number={1} title="Install the theme components">
  Use the shadcn CLI to install components from the DF Imóveis registry path.
</Step>

```bash
npx shadcn@latest add "https://registry.blyk.design/r/df-imoveis/indicator-card.json"
npx shadcn@latest add "https://registry.blyk.design/r/df-imoveis/imovel-card.json"
```

<Step number={2} title="Add the theme CSS">
  Copy the DF Imóveis token overrides into your project. These override the base blyk tokens with the brand palette.
</Step>

```css
[data-theme="df-imoveis"] {
  --primary: 214.8 64.8% 24.5%;
  --primary-foreground: 210 40% 98%;
  --secondary: 210 40% 96.1%;
  --secondary-foreground: 222.2 47.4% 11.2%;
  /* ... see registry/df-imoveis/tokens/ for the full list */
}
```

<Step number={3} title="Activate the theme">
  Apply the theme using a data attribute on the root element.
</Step>

```html
<html data-theme="df-imoveis">
```
