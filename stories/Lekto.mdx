## Installation

<Step number={1} title="Install the theme components">
  Use the shadcn CLI to install components from the Lekto registry path.
</Step>

```bash
npx shadcn@latest add button --registry https://registry.blyk.design/r/lekto
npx shadcn@latest add --all --registry https://registry.blyk.design/r/lekto
```

<Step number={2} title="Add the theme CSS">
  Copy the Lekto token overrides into your project. These override the base blyk tokens with the brand palette.
</Step>

```css
[data-theme="lekto"] {
  --primary: 287 72% 38%;
  --primary-foreground: 285 100% 98%;
  --font-sans: "Mulish", ui-sans-serif, system-ui, sans-serif;
  --font-mono: "Geist Mono", ui-monospace, monospace;
  /* ... see registry/lekto/tokens/ for the full list */
}
```

<Step number={3} title="Activate the theme">
  Apply the theme using a data attribute on the root element.
</Step>

```html
<html data-theme="lekto">
```
