## Identidade visual

O tema BIUD Sebrae utiliza uma paleta baseada em azul (#1B50E2) como cor primária, combinada com tons de slate para backgrounds e bordas. A tipografia principal é a **Figtree**, com **Geist Mono** para código.

## Instalação

<Step number={1} title="Instalar componentes do tema">
  Use o shadcn CLI para instalar componentes a partir do registry BIUD Sebrae.
</Step>

```bash
# Instalar um componente específico
npx shadcn@latest add button --registry https://registry.blyk.design/r/biud-sebrae

# Instalar todos os componentes
npx shadcn@latest add --all --registry https://registry.blyk.design/r/biud-sebrae
```

<Step number={2} title="Adicionar os tokens do tema">
  Copie os overrides de tokens do BIUD Sebrae para o seu projeto. Os arquivos estão em `registry/biud-sebrae/tokens/`.
</Step>

```css
[data-theme="biud-sebrae"] {
  --primary: 224 76% 50%;
  --primary-foreground: 210 40% 99%;
  --font-sans: "Figtree", ui-sans-serif, system-ui, sans-serif;
  --font-mono: "Geist Mono", ui-monospace, monospace;
  /* ... veja registry/biud-sebrae/tokens/ para a lista completa */
}

[data-theme="biud-sebrae"].dark {
  --primary: 214 32% 91%;
  --primary-foreground: 224 76% 50%;
  /* ... */
}
```

<Step number={3} title="Carregar as fontes">
  Inclua as fontes Figtree e Geist Mono no seu projeto para que a tipografia seja exibida corretamente.
</Step>

```html
<!-- Figtree (Google Fonts) -->
<link rel="preconnect" href="https://fonts.googleapis.com">
<link href="https://fonts.googleapis.com/css2?family=Figtree:wght@400;500;600;700&display=swap" rel="stylesheet">
```

```css
/* Ou via CSS */
@import url('https://fonts.googleapis.com/css2?family=Figtree:wght@400;500;600;700&display=swap');
```

<Step number={4} title="Ativar o tema">
  Aplique o tema usando o atributo `data-theme` no elemento raiz.
</Step>

```html
<html data-theme="biud-sebrae">
```

## Estrutura do tema

```
registry/biud-sebrae/
├── README.md           ← documentação técnica
├── tokens/
│   ├── colors.css     ← cores light/dark
│   ├── typography.css ← fontes
│   └── radius.css     ← border-radius
└── components/
    └── ui/            ← overrides de componentes (opcional)
```

Os arquivos de tokens são mesclados com o tema base durante o build. Sobrescreva apenas o que for necessário; o restante herda do blyk-base.
