## Installation

<Step number={1} title="Install the theme components">
  Use the shadcn CLI to install components from the BIUD Sebrae registry path.
</Step>

```bash
npx shadcn@latest add button --registry https://registry.blyk.design/r/biud-sebrae
npx shadcn@latest add --all --registry https://registry.blyk.design/r/biud-sebrae
```

<Step number={2} title="Add the theme CSS">
  Copy the BIUD Sebrae token overrides into your project. These override the base blyk tokens with the brand palette.
</Step>

```css
[data-theme="biud-sebrae"] {
  --primary: 224 76% 50%;
  --primary-foreground: 210 40% 99%;
  --font-sans: "Figtree", ui-sans-serif, system-ui, sans-serif;
  --font-mono: "Geist Mono", ui-monospace, monospace;
  /* ... see registry/biud-sebrae/tokens/ for the full list */
}
```

<Step number={3} title="Activate the theme">
  Apply the theme using a data attribute on the root element.
</Step>

```html
<html data-theme="biud-sebrae">
```
