## Quick Start

<Step number={1} title="Install the shadcn CLI">
  Make sure you have a React project with Tailwind CSS v4 configured.
</Step>

```bash
npx shadcn@latest init
```

<Step number={2} title="Add components from the registry">
  Install any component directly from the blyk registry.
</Step>

```bash
npx shadcn@latest add "https://registry.blyk.design/r/base/button.json"
```

<Step number={3} title="Use in your project">
  Import and use the component. The source code lives in your project.
</Step>

```tsx
import { Button } from "@/components/ui/button"

export function MyPage() {
  return <Button variant="outline">Get Started</Button>
}
```

<Tip>
  You can install multiple components at once by passing multiple registry URLs to the CLI.
</Tip>

## Theming

blyk uses **CSS variables** for theming. The base theme ships with light and dark mode support out of the box.

Custom themes can be applied by overriding the CSS variables with a `data-theme` attribute:

```css
[data-theme="my-brand"] {
  --primary: 220 90% 56%;
  --primary-foreground: 0 0% 100%;
  /* ... */
}
```

<div style={{ display: "flex", gap: "0.5rem", margin: "1rem 0" }}>
  {[
    { bg: "--primary", fg: "--primary-foreground", label: "Primary" },
    { bg: "--secondary", fg: "--secondary-foreground", label: "Secondary" },
    { bg: "--accent", fg: "--accent-foreground", label: "Accent" },
    { bg: "--muted", fg: "--muted-foreground", label: "Muted" },
    { bg: "--destructive", fg: "--destructive-foreground", label: "Destructive" },
  ].map((item) => (
    <div
      key={item.label}
      style={{
        flex: 1,
        backgroundColor: `hsl(var(${item.bg}))`,
        color: `hsl(var(${item.fg}))`,
        padding: "1rem 0.75rem",
        borderRadius: "var(--radius)",
        textAlign: "center",
        fontSize: "0.75rem",
        fontWeight: 500,
      }}
    >
      {item.label}
    </div>
  ))}
</div>
